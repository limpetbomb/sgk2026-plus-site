# SGK+ Petit Only Announcement Site

高石タケル x 本宮大輔 有志プチオンリー「SGK+」の告知サイトです。

## Project

- イベント名: SGK+
- 内容: タケ大有志プチオンリー告知サイト
- 開催日: 2026年12月20日（日）
- 親イベント: DOZEN ROSE FES 2026
- 関連オンリー: 希望を駆ける白き翼 DR2026
- 会場: 東京ビッグサイト
- 主催・協力: おけつ・わたる・りむこ

このサイトは有志による非公式企画の告知サイトであり、公式および関係者各所とは一切関係ありません。

## Site Purpose

- 参加サークル募集
- 一般参加者への案内
- 当日企画の告知
- サークル一覧の掲載
- 公式X・問い合わせ導線の設置

## Design Direction

- 同人イベント告知サイトらしいポスター感
- かわいい、サブカル、韓国っぽさ
- 企業サイトっぽくしすぎない
- ベースカラーは濃い水色
- 文字色は濃い紺
- 黄色はアクセントとして使用
- 白い紙・点線・マスキングテープ風ラベルを適度に使う
- 情報量が多くても読みやすくする
- スマホで見やすいレスポンシブ対応

## Current Visual Notes

- 中央に縦長カラムがある構成
- 背景は水色系で統一
- 見出し画像はユーザー提供画像を使用
- 開催概要は白い角丸パネル内に情報を配置
- 開催概要の黄色ラベルは、マスキングテープ風に少し斜め配置
- NEWSは点線区切り
- 参加方法は「一般参加」「サークル参加」を上下に表示
- サークル一覧はトップでは名簿風プレビュー、一覧ページではカード型
- 当日企画は黄色の角丸カードを3つ並べる

## Files

- `index.html`: トップページ
- `about.html`: イベント概要
- `join.html`: 参加方法
- `circles.html`: サークル一覧
- `plans.html`: 当日企画
- `news.html`: お知らせ
- `contact.html`: 問い合わせ
- `assets/css/style.css`: 全体スタイル
- `assets/js/main.js`: メニュー・スクロール演出
- `assets/images/`: ロゴ、見出し、SD画像など

## Editing Notes

- まず `index.html` と `assets/css/style.css` を編集することが多いです。
- 色はCSS変数で管理しています。
- 画像差し替えは `assets/images/` 内のファイルを置き換えると楽です。
- 公式Xアカウント、参加表明フォーム、問い合わせフォームURLは未作成のため仮導線です。
- 文言は仮の箇所があります。公開前に主催側で確認してください。

## GitHub Pages

静的サイトなのでGitHub Pagesで公開できます。

1. このフォルダの中身をGitHubリポジトリ直下に置く
2. GitHubの `Settings` -> `Pages` を開く
3. `Deploy from a branch` を選択
4. Branchを `main`、Folderを `/root` にする
5. 表示されたURLで確認する
